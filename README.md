
# 🎬 Movie Ticket Booking Application

A full-stack Movie Ticket Booking Application that allows users to browse movies, select seats, and book tickets seamlessly. This project demonstrates core concepts of web development and backend logic using Java.

---

## 🚀 Features

- 🎥 Browse available movies
- 🕒 View show timings
- 💺 Seat selection system
- 🧾 Ticket booking functionality
- 👤 User input handling
- 📊 Basic booking management

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3

### Backend
- Java (Core Java)
